# Login Your AWS Management Console by your aws account

For Mac OS X

## download

```
$ git clone https://github.com/uji52/awsmc.git
```

## awsmc.json

### RoleArn
Policy Create Role

### RoleSessionName
sessionName

### DurationSeconds
Session Exp (minutes)

### Policy
Policy you want to request

## start
```
$ npm install
$ node awsmc
```
